Safari / iOS Instructions

How to install Restore Iran Flag on Safari

1

Install Userscripts App

Download the free Userscripts app from the App Store. It works on iPhone, iPad, and Mac.

2

Enable the Extension

On iPhone/iPad: Go to Settings → Safari → Extensions → Userscripts → Enable it and allow on "All Websites"

On Mac: Go to Safari → Settings → Extensions → Enable Userscripts

3

Set Script Directory

Open the Userscripts app and tap "Set Userscripts Directory". Create or select a folder (e.g., in iCloud Drive or On My iPhone).

4

Download the Script

Download our userscript file and save it to the directory you set in step 3.

Download Userscript
5

Activate

Open Safari, go to x.com or twitter.com. Tap the puzzle icon in the address bar, then tap Userscripts. The script should be listed and active.

Tip: If you don't see the flag change, refresh the page. The script runs automatically on Twitter/X.
